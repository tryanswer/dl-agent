# DriftLedger MCP

This directory is reserved for a future MCP server or manifest package that
exposes DriftLedger backend atoms as tools.

The CLI remains the MVP integration surface. Add MCP files only when the tool
contract is stable enough to support Codex, Claude Code, OpenClaw, and other
agent runtimes without shell command wrapping.

Planned tool groups:

- Workspace and auth context inspection.
- Metadata, source binding, raw dataset, and assembled dataset operations.
- Reconciliation model and rule management.
- RuleForest build and status operations.
- Run submission, execution, result indexes, and incident retrieval.
